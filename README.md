# 🌦️ SkyCast Weather

A sleek, responsive weather application using the WeatherAPI. 

## 🚀 [View Live Demo](INSERT_YOUR_GITHUB_PAGES_LINK_HERE)

### 🛠️ Key Features
- **Dynamic Backgrounds:** Changes based on day/night cycles.
- **Geolocation:** Automatically detects your local weather.
- **Dual Units:** Toggle between Celsius and Fahrenheit.
- **Air Quality Index (AQI):** Real-time pollution data.

### 💻 Technologies Used
- HTML5, CSS3 (Custom Properties & Grid)
- JavaScript ES6 (Async/Await, Fetch API)
- WeatherAPI.com
