# 🌦️ Advanced Weather App

A modern weather application built with HTML, CSS, and JavaScript that provides real-time weather information with a clean glassmorphism user interface.

## ✨ Features

* Search weather by city name
* Real-time weather data
* Temperature display with weather icons
* Humidity information
* Wind speed details
* UV Index display
* Air Quality Index (AQI)
* Multi-day weather forecast
* Celsius (°C) and Fahrenheit (°F) toggle
* Loading animation
* Responsive design for all devices
* Modern Glassmorphism UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Weather API
* Google Fonts (Poppins)

## 🚀 How It Works

1. Enter a city name.
2. Click the Search button.
3. The application fetches weather data from the API.
4. Current weather details and forecast are displayed instantly.
